About This Site

Welcome to the official GitHub page dedicated to hosting Geographic Information Systems (GIS) applications for the North Carolina National Guard (NCNG). This platform provides secure and reliable access to web-based GIS applications designed to enhance operational planning, situational awareness, resource management, and emergency response capabilities.

Purpose and Benefits

  • Centralized Platform: Convenient access to critical GIS applications.
  • Operational Support: Interactive tools for strategic and tactical planning.
  • Collaboration: Facilitate sharing of geospatial resources across NCNG.
  • Real-Time Information: Up-to-date spatial analysis and data visualization.
